สนับสนุน

เริ่มต้นใช้งาน

การ คําแนะนําการเริ่มต้นใช้งาน ให้คําถามแบบโต้ตอบเพื่อเชื่อมโยงคุณกับแหล่งข้อมูลที่สามารถช่วยให้คุณเริ่มต้นได้

รับการสนับสนุนเพิ่มเติม

คําแนะนําในการรับการสนับสนุนเพิ่มเติมสําหรับ ชุดระบบอัตโนมัติสําหรับ Power Platform:

  • อ้างอิงถึงเรา คําติชมและการสนับสนุน หน้าสําหรับคําแนะนําทั่วไป

  • ใช้ของเรา เรียนรู้ชุดระบบอัตโนมัติ หน้าเพื่อทําตามขั้นตอนการติดตั้งและคําแนะนําการใช้งาน

  • ใช้ปุ่ม ปัญหาเกี่ยวกับชุดระบบอัตโนมัติ เทมเพลตสําหรับการค้นหาข้อบกพร่องที่รู้จัก / ช่องว่างของคุณลักษณะ หากคุณไม่พบปัญหาที่ตรงกัน ให้มองหาปัญหาใหม่โดยใช้เทมเพลตที่ให้มา

  • ใช้ของเรา [วิดีโอแนะนําวิธีใช้](n แต่ละเหตุการณ์สําคัญเนื่องจากข้อบกพร่องหรือคุณสมบัติที่มีลําดับความสําคัญ) เพื่อช่วยคุณในการแสดงวิธีการทํางานของกระบวนการติดตั้ง

  • เข้าร่วมกับเรา เวลาทําการ เพื่อถามคําถามเกี่ยวกับปัญหาทั่วไปหรือแผนงานผลิตภัณฑ์

  • ติดต่อ Microsoft Global Black Belt, สถาปนิกโซลูชันระบบคลาวด์ หรือช่องทางการสนับสนุนปกติของคุณสําหรับคําแนะนําเพิ่มเติม

ทีมพาวเวอร์แคท

ทีม Power CAT หลักไม่สามารถดําเนินการแก้ไขปัญหาหรือให้คําปรึกษาที่กําหนดเองกับผู้ชมในวงกว้างได้ หากคุณกําลังประสบปัญหาหรือมีคําถามเกี่ยวกับ ชุดระบบอัตโนมัติสําหรับ Power Platform, เข้าร่วมกับเรา เวลาทําการ หรือแจ้งปัญหาหรือคําถามของคุณผ่านทาง https://aka.ms/ak4pp-issues
