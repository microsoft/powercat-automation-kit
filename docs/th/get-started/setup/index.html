ตั้ง ค่า

ใช้ขั้นตอนด้านล่างเพื่อช่วยคุณในการสร้างการกําหนดค่าสําหรับสภาพแวดล้อมของคุณเพื่อให้คุณสามารถติดตั้งและกําหนดค่าการติดตั้งของคุณได้อย่างรวดเร็ว

เริ่มต้นใช้งานโดยทําตามขั้นตอนใน ติดตั้ง เพื่อดาวน์โหลดคอมโพเนนต์การติดตั้งบรรทัดคําสั่งรุ่นล่าสุด

หากคุณต้องการให้คําติชมเกี่ยวกับกระบวนการตั้งค่า ถ้าเป็นเช่นนั้นคุณสามารถใช้ คําติชมเกี่ยวกับการตั้งค่า หน้า
