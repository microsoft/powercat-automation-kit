Automation Kit for Power Platform
ชุดระบบอัตโนมัติสําหรับ Power Platform

การตั้งชื่อ

ข้างเดือนและหมายเลขเวอร์ชัน ชุดระบบอัตโนมัติสําหรับ Power Platform ใช้แท็กชื่อรหัสในแต่ละรุ่น

กระบวนการสร้าง

  1. ติดตั้งการขึ้นต่อกันของโหนด
npm install @criblinc/docker-names
  1. เพิ่มตัวอย่างโหนต่อไปนี้เป็นแฟ้มชื่อ index.mjs
import {generateName} from '@criblinc/docker-names'

const dockerName = generateName();

console.log(dockerName);
  1. เรียกใช้ตัวอย่าง
node index.mjs
