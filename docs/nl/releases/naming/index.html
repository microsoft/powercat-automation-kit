Naamgeving

Naast de maand- en versienummers de Automation Kit voor Power Platform gebruikt elke release een codenaamtag.

Generatieproces

  1. Knooppuntafhankelijkheden installeren
npm install @criblinc/docker-names
  1. Het volgende knooppuntvoorbeeld toevoegen als bestand met de naam index.mjs
import {generateName} from '@criblinc/docker-names'

const dockerName = generateName();

console.log(dockerName);
  1. Het voorbeeld uitvoeren
node index.mjs
