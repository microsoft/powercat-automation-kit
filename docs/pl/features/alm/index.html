Zarządzanie cyklem życia aplikacji (ALM)

Ta strona zawiera omówienie składników, które mogą pomóc w korzystaniu z ALM z zestawem Automation Kit dla przepływów pracy programu Power Automate Desktop zawartych w Rozwiązania Power Platform.

Pełny ekran

Streszczenie

Szukając rozwiązania ALM Power Platform zawierające składniki programu Power Automate Desktop

  1. Zapoznaj się z funkcjami potoków Power Platform środowiska zarządzanego, aby skorzystać z funkcji produktu w skali przedsiębiorstwa w celu zarządzania rozwiązaniami w środowiskach i zarządzania nimi.

  1. W razie potrzeby zbadaj Narzędzia Microsoft Power Platform Build Tools dla usługi Azure DevOps, Akcje usługi GitHub dla usługi Microsoft Power Platform lub Power Platform CLI w celu integracji i automatyzacji procesów ALM DevOps.

  1. Rozważ użycie Akcelerator ALM dla Power Platform. Akcelerator ALM udostępnia wstępnie utworzony zestaw szablonów usługi Azure DevOps, który automatyzuje wiele Power Platform zadań ALM przy użyciu zintegrowanego nadzoru kontroli źródła.

Nauka z Power CAT

Możesz również przeczytać więcej, w jaki sposób my, jako zespół Power CAT, używamy ALM Accelerator do dostarczania Zestaw automatyki Power CAT ALM.

Mapa drogowa

Zespół zestawu Automation Kit współpracuje z zespołem ALM Accelerator, aby dodać zadania specyficzne dla programu Power Automate Desktop do istniejących szablonów, które obejmują:

  • Porównanie definicji programu Power Automate Desktop obok siebie.

  • Sprawdzanie reguł sprawdzania poprawności dla programu Power Automate Desktop.

  • Wykonanie testów jednostkowych, integracyjnych i systemowych w ramach potoku CI/CD.
