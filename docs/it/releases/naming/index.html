Kit di automazione per Power Platform

Denominazione

Accanto ai numeri di mese e di versione il Kit di automazione per Power Platform utilizza un tag code name ogni versione.

Processo di generazione

  1. Installare le dipendenze dei nodi
npm install @criblinc/docker-names
  1. Aggiungere il seguente esempio di nodo come file denominato index.mjs
import {generateName} from '@criblinc/docker-names'

const dockerName = generateName();

console.log(dockerName);
  1. Eseguire l’esempio
node index.mjs
