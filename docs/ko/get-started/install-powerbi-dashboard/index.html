설치 자동화 키트 Power BI 대시보드

자동화 우수 센터에 대한 Power BI 대시보드를 시작하려면 주 환경(자동화 CoE Main이 설치된 위치)에 대한 환경 URL이 필요합니다.

에서 다운로드한 Power BI 템플릿을 시작할 때 환경 URL을 입력합니다. 전력 플랫폼용 자동화 키트 릴리스
