Centre d’excellence en automatisation

Question: Comment le Kit d’automatisation pour Power Platform liés à la Kit CdE?

Il s’agit de solutions complémentaires que votre équipe de centre d’excellence peut installer pour vous aider à gérer votre environnement Low Code.

Question: Pourquoi le Kit d’automatisation pour Power Platform pas un autre module du Kit CdE?

Le Kit d’automatisation pour Power Platform s’est concentré sur le Cycle de vie de l’automatisation. Comme l’indique le Centre d’excellence en automatisation Chacun des kits répond aux besoins des différentes composantes et activités de l’ensemble du processus.

Question: Le Kit d’automatisation pour Power Platform et Kit CdE être utilisés ensemble?

Absolument, l’accélérateur ALM du kit CoE est un exemple de déploiement et d’intégration de Power Automate Desktop avec le Kit d’automatisation pour Power Platform sont une combinaison naturelle pour automatiser l’intégration avec le contrôle de code source et automatiser le déploiement entre les environnements.
