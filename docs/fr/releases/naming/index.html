Kit d’automatisation pour Power Platform

Nommage

À côté des numéros de mois et de version, le Kit d’automatisation pour Power Platform utilise une balise de nom de code pour chaque version.

Processus de génération

  1. Installer les dépendances de nœud
npm install @criblinc/docker-names
  1. Ajouter l’exemple de nœud suivant en tant que fichier nommé index.mjs
import {generateName} from '@criblinc/docker-names'

const dockerName = generateName();

console.log(dockerName);
  1. Exécuter l’exemple
node index.mjs
