若要启动自动化卓越中心的 Power BI 仪表板，需要主环境（安装自动化 CoE Main 环境的位置）的环境 URL。

若要获取环境 URL，请导航到Power Platform管理中心，转到“环境”，选择环境，然后记下环境 URL：

![获取环境屏幕 - Power Platform管理中心](/images/get-environment.png)
